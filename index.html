Index of Pages

This is a non-exhaustive, incomplete list of the projects I've made with Wes Bos's Javascript 30 course.

The idea of the course is to use only vanilla Javascript and to get acquainted with the things that modern JS can do without having to resort to frameworks or libraries (looking at you, jQuery).

Most of the projects below have some dynamic element to them, but a couple are just programmatic exercises. They're up here for completion's sake.

Speaking of completion's sake, a handful of projects here involve running a server, and since I'm just uploading these as static HTML pages, they don't run properly. They involved things like taking control of the user's webcam and feeding it into a canvas to manipulate it, using Javascript's built-in speech-to-text abilities, and using geolocation. For more info go check out the course itself.

  • Drum Kit

    A keyboard based drum kit, loading audio with <audio> tags

  • Javascript Clock

    An analogue clock created with <div>s and animated with Javascript

  • CSS Variables

    Using sliders to adjust CSS variable values dynamically

  • Array Methods 1

    Practicing a bunch of array methods and ES6 function declarations, i.e. () => { console.log('This guy') }. Gonna be frank, not much to see here

  • Flexbox Practice

    Adjustable panels with pictures and text using Flexbox

  • AJAX Requests

    Grabbing a bunch of cities from an endpoint elsewhere and using them to populate and array after page load. Very simplistic AJAX requests here

  • Array Methods 2

    Like Array Methods 1, not much to see here, just some practice with array methods

  • HTML5 Canvas

    A canvas you get to draw on with a pen that changes color and width wildly

  • Shift-Click Checkboxes

    Little tool for being able to shift-click checkboxes so that all intermediate checkboxes also highlight, like in a native filesystem

  • HTML5 Video

    Using Javascript to change up the appearance of the standard HTML5 video player

  • Key Detection

    Detects what keys you've been pressing and matches them up against a password. If you're right, you win a little surprise

  • Image Slide-Ins

    Images sliding in as you scroll onto them

  • References and Copies in JS

    Not much to see here, just a bit of a demonstration of the way that Javascript treats copies of/references to variables

  • Local Storage

    To-do list employing localStorage to persist data on page load

  • Text-Shadow with the Mouse

    Text shadow on a few words that follows the mouse, as if the mouse were a flashlight

  • Sorting Band Names with Articles

    Not exactly dynamic, a demonstration of how to sort a list of items independent of leading articles (the, a, an)

  • Calculating Total Time

    Given a list of items with data-time attributes, how to pull the value from that attribute and add each node up to yield a total time

  • Movable Link Highlight

    A highlight behind links that follows your cursor around the page as you mouse over links

  • Text to Speech

    Type out some words and let the computer turn those words into text for you. Especially fun if you type in a language that isn't the language in which the computer reads it out

  • Sticky Nav

    A nav bar that sticks with the page once you scroll past it. Also displays a little tab with a logo or company name or product title or something, if you want
